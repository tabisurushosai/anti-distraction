# ストア掲載情報 - anti-distraction

## 名前
Anti-Distraction — 集中モードでサイト遮断

## 概要 (132字)
SNS や動画サイトを集中モード中に遮断。タイマーと連動して作業中の誘惑を断つ。クールダウン期間設定可。
Block SNS/video sites during focus mode. Timer-linked distraction blocker with cooldown.

## 詳細説明
作業中の SNS や動画サイトの誘惑を断つ Chrome 拡張機能です。

【主な機能】
- ブロックするサイトをカスタマイズ可
- 集中モードの時間設定 (15分〜数時間)
- クールダウン期間 (一度設定したら短時間は解除不可)
- ホワイトリスト機能
- 集中モード時のブロック画面 (励ましのメッセージ表示)
- 統計機能 (今週の集中時間を記録)
- 完全オフライン、データ送信なし

【こんな人にオススメ】
- SNS のスクロールが止まらない方
- YouTube を見始めると終わらない方
- 試験勉強や締切が迫っている方
- リモートワークで集中したい方

【プライバシー】
データは端末内のみ。広告なし。完全無料 (Premium のみ $3 買い切り)。
