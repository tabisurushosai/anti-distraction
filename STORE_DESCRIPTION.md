# ストア掲載情報 - Anti-Distraction

## 日本語

### 名前

Anti-Distraction — SNS・動画の利用制限

### 概要

SNSや動画サイトに利用時間の上限を設定。解除クールダウンと利用統計で作業への集中を支援します。

### 詳細説明

SNSや動画サイトの利用時間を制限し、作業への集中を支援するChrome拡張機能です。

【主な機能】
- 制限するサイトをカスタマイズ
- 1日と1セッションの利用時間を設定
- 対象サイトをグレースケール表示
- 一時解除後のクールダウン
- 最近7日間の利用統計
- 設定とサイト利用時間は端末内だけに保存

【こんな人にオススメ】
- SNS のスクロールが止まらない方
- YouTube を見始めると終わらない方
- 試験勉強や締切が迫っている方
- リモートワークで集中したい方

【プライバシー】
閲覧履歴は収集しません。設定とサイト利用時間は端末内だけに保存し、広告は表示しません。
Premiumは$9買い切りで、30日統計、連続達成日数、サイト数無制限、クールダウン設定、
一時解除回数の上限拡張を利用できます。購入確認時だけ、公開された商品IDと入力した
ライセンスキーをGumroadへ送信します。

## English

### Name

Anti-Distraction — Site Time Limiter

### Summary

Block distracting social and video sites with focus limits, cooldowns, and local activity stats.

### Detailed description

Anti-Distraction is a Chrome extension that helps you stay focused by limiting
access to distracting social and video sites while you work.

Key features:

- Choose which sites to limit.
- Set daily and per-session time limits.
- Use a cooldown before temporarily unblocking a site.
- See your recent focus activity.
- Keep browsing settings and usage statistics on your device.

Premium is a one-time $9 purchase that adds 30-day statistics, streaks,
unlimited sites, customizable cooldowns, and higher temporary-unblock limits.
Payment is processed by Gumroad. Only the public product ID and the license key
you enter are sent to Gumroad for purchase verification.

Anti-Distraction has no advertising. It does not send browsing URLs, blocked
site lists, or usage statistics to Gumroad.
